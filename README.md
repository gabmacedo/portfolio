# Gabriel Macedo | Portfólio

Site pessoal moderno feito com React, Vite, TypeScript e TailwindCSS.

## Rodar o projeto

```bash
npm install
npm run dev
```

Depois, abra o endereço exibido no terminal.

## Links fáceis de editar

As URLs externas ficam em `src/constants/links.ts`:

- `LINKTREE_URL`
- `RESUME_URL`
- `LINKEDIN_URL`

## Estrutura principal

```text
src/
  components/
    About.tsx
    Contact.tsx
    Header.tsx
    Hero.tsx
    Projects.tsx
  constants/
    links.ts
  data/
    projects.ts
  styles/
    index.css
  App.tsx
  main.tsx
```
